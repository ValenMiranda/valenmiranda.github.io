HERRAMIENTAS

HTML

sigla en inglés de HyperText Markup Language, hace referencia al lenguaje de marcado para la elaboración de páginas web.

JAVASCRIPT

(abreviado comúnmente JS) es un lenguaje de programación interpretado, dialecto del estándar ECMAScript.

JQUERY

Es una biblioteca multiplataforma de JavaScript.

CSS

Sigla en nglés de Cascading Style Sheets, es el lenguaje utilizado para describir la presentación de documentos HTML o XML.

SASS

Sigla en inglés de Syntactically Awesome Stylesheets, es un lenguaje de hoja de estilos.

GULP

Es un conjunto de herramientas de JavaScript de código abierto, utilizado como un sistema de generación de streaming.

BOOTSTRAP

Es un framework o conjunto de herramientas de Código abierto para diseño de sitios y aplicaciones web.

MATERIALIZE

Es un framework o conjunto de herramientas de Código abierto para diseño de sitios y aplicaciones web.

SLACK

Es una herramienta de comunicación en equipo.

PHOTOSHOP

Es un editor de gráficos rasterizados.

ILLUSTRATOR

Es un editor de gráficos vectoriales en forma de taller de arte.

INDESIGN

Es una aplicación para la composición digital de páginas.
